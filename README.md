# 텍스트 꾸미기 

**GitHub**는 *원격 저장소*를 제공하는 ~서비스~이기 때문에 

온라인 상에서 Gi의 ***버전 관리 기능***을 사용할 수 있습니다. 

한 줄 짜리 소스 코드는 ` fuction add(x,y) {return x+y; };` 처럼 사용합니다.

깃허브 데브를 사용해 문서를 수정 중입니다.
